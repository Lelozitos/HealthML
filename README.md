## ‚öôÔ∏èÔ∏èÔ∏èÔ∏èÔ∏è Installation

1. Clone the repo

   ```sh
   git clone https://github.com/Lelozitos/HealthML.git
   ```

1. Install Requirements

   - Open `InstallRequirements.bat` or
   - Install with pip
     ```sh
     python -m pip install requirements.txt
     ```

   ```sh
   python ./main.py
   ```
